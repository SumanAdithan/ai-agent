# Where LangChain Fits in this Stack

**LangChain** is a massive toolkit designed for building LLM workflows. Its primary purpose is to provide consistent, abstracted wrappers over different models and tools so that developers can write code once and seamlessly swap out underlying providers.

## Key Features
*   **Model Wrappers:** Switch between OpenAI, Gemini, and Groq with zero structural code changes.
*   **Tool Integrations:** Built-in wrappers for vector databases, web scrapers, and APIs.
*   **LCEL:** LangChain Expression Language for building declarative pipelines (Chains).
*   **RAG:** Robust components for Retrieval-Augmented Generation.

## Architecture Placement

LangChain sits perfectly inside the **Orchestration** layer of a modern AI application.
