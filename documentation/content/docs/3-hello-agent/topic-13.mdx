# Project Setup & Dependencies

Before we build our first agent, we need to initialize a fresh Node.js environment configured with TypeScript, LangChain packages, and environment variable support.

<Steps>

<Step>
### Initialize Project
Initialize your project using `pnpm init` and ensure you set `"type": "module"` in your `package.json` to use modern ES imports natively.
</Step>

<Step>
### Install Dependencies
Install the necessary raw provider SDKs (OpenAI and Google GenAI) and typescript tooling.

```bash
pnpm add dotenv @google/genai openai
pnpm add -D typescript @types/node tsx
```
</Step>

<Step>
### Project Structure
We keep the architecture simple. An `src` folder for logic, and root configs for typescript and environment variables.

  <Folder name="hello-agent" defaultOpen>

</Files>
</Step>

<Step>
### TypeScript Configuration
We configure TypeScript strictly for Modern ECMAScript (ES2022/ESNext) and use node module resolution.

```json title="tsconfig.json"
{
  "compilerOptions": {
    "target": "es2022",
    "module": "esnext",
    "moduleResolution": "bundler",
    "strict": true,
    "esModuleInterop": true,
    "resolveJsonModule": true,
    "skipLibCheck": true,
    "noEmit": true,
    "types": ["node"]
  },
  "include": ["src"]
}
```
</Step>

</Steps>
