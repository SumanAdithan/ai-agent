# What are we building

In this module, we are building our first, foundational project: **Hello Agent**.

Before diving into complex agentic workflows, memory, and tool usage, we must understand the core primitive of all AI agents: **The LLM Provider Layer**.

This project will teach you how to:
1. Securely load API keys for different platforms.
2. Initialize and configure bare SDKs (like `@google/genai` and `openai`).
3. Build a dynamic routing system that allows you to easily swap between Gemini, OpenAI, and Groq on the fly.
4. Normalize the outputs into a standard, predictable JSON format.

By the end of this module, you will have a rock-solid foundation for requesting generations from any major language model.
